Get Started → View on GitHub

Six Powerful Tools

Everything you need for professional-grade UI improvements

improve_ui

Sends your frontend code to Gemini for a full aesthetic overhaul. Fixes spacing, typography, color contrast, hover states, and accessibility.

analyze_ui

Returns a structured JSON design review with a score (0–100), issues list, and actionable fixes.

create_component

Create a brand-new component or page from a text description. Production-ready, accessible, responsive.

generate_design_tokens

Generates a complete CSS variable system with colors, spacing, typography, shadows, transitions, and z-index.

ask_gemini_design

Free-form design Q&A. Ask Gemini anything about UI/UX — font pairings, color theory, layout patterns.

visual_diff

Generate a side-by-side HTML visual diff report. See before/after changes with improvement scores.
